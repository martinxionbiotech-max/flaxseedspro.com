# Flaxseed Powder

**Product Code:** FSP-PDR-001 (Full-fat), FSP-PDR-002 (Defatted)  
**Revision:** V11.0

---

## Description

Flaxseed powder is produced by milling clean flaxseed kernel or whole seed. Two grades are available: **full-fat** (native oil retained) and **defatted** (oil removed by pressing prior to milling). Particle size is controlled across six mesh grades for application-specific performance.
