# Processing Standards

---

## Standard Operating Procedures

### 1. Raw Material Receiving

- All inbound lots sampled and tested before unloading
- Moisture, protein, fat, and visual inspection on every lot
- Reject criteria: moisture > 10%, visible mold, live insect infestation
- Organic lots segregated and verified against certification docs

### 2. Cleaning Parameters

| Parameter | Set Point | Tolerance |
|:----------|:---------:|:---------:|
| Gravity separator deck angle | 7° | ±0.5° |
| Airflow velocity | 3.5 m/s | ±0.3 m/s |
| Optical sorter sensitivity | 95% | ±2% |
| Magnetic separator strength | 10,000 Gauss | minimum |

### 3. Milling Parameters

| Parameter | Set Point | Tolerance |
|:----------|:---------:|:---------:|
| Hammer mill RPM | 3,600 | ±100 |
| Screen size (fine) | 1.0 mm | ±0.1 mm |
| Screen size (medium) | 2.0 mm | ±0.2 mm |
| Product temperature | < 45°C | maximum |
| Throughput | 2 MT/hr | ±0.2 MT/hr |

### 4. Cold Press Extraction

| Parameter | Set Point | Tolerance |
|:----------|:---------:|:---------:|
| Screw speed | 15 RPM | ±2 RPM |
| Barrel temperature | < 40°C | maximum |
| Nozzle diameter | 8 mm | ±0.5 mm |
| Press cake thickness | 6 mm | ±1 mm |

### 5. Packaging Standards

- Nitrogen flush: residual O₂ < 2%
- Bag weight accuracy: ±10g (20kg bag)
- Seal integrity: 100% inline inspection
- Metal detection: 1.0 mm ferrous, 1.5 mm non-ferrous
- Label accuracy: 100% scanned verification

## Sanitation Standards

- Master sanitation schedule: weekly rotation
- Allergen cleaning: full wet clean between production runs
- Environmental monitoring: 20 swab points per week
- Pest control: 3rd party contractor, monthly service
