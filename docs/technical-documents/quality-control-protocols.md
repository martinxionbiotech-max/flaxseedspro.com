# Quality Control Protocols

---

## QC Testing Schedule

### Raw Material (Every Lot)

| Test | Method | Acceptance Criteria |
|:-----|:-------|:-------------------|
| Moisture | AOAC 925.10 | ≤ 8.0% |
| Protein | AOAC 984.13 | 18-24% |
| Fat | AOAC 920.39 | 38-44% |
| Visual | Manual inspection | No mold, insects, off-colors |
| Foreign material | Sieve + handpick | ≤ 0.1% |
| Organoleptic | Sensory panel | Typical flavor, no rancidity |

### In-Process (Every 2 Hours)

| Test | Method | Acceptance Criteria |
|:-----|:-------|:-------------------|
| Moisture | NIR rapid | Within spec ±0.5% |
| Particle size | Sieve analysis | Within target ±5% |
| Temperature | IR thermometer | < 45°C (milling), < 40°C (oil) |
| Metal detector | Test wand | Functional check |

### Finished Product (Every Batch)

| Test | Method | Acceptance Criteria |
|:-----|:-------|:-------------------|
| Full nutritional | NIR + wet chemistry | Meets spec sheet |
| Microbiological | AOAC | See product specs |
| Packaging integrity | Visual + seal test | 100% pass |
| Label verification | Scanner | 100% match |
| Net weight | Check-weigher | ±10g (bags), ±5g (retail) |
| Shelf life | Accelerated testing | Confirmed per product |

## Certificate of Analysis (COA)

Every batch ships with a COA including:
- Product name, lot number, production date
- Moisture, protein, fat, fiber results
- ALA omega-3 percentage
- Microbiological results
- Peroxide value (oil products)
- Organic certification reference (organic products)

## Non-Conformance Procedure

1. **Detection** — QC flags out-of-spec result
2. **Hold** — Affected product placed on hold status
3. **Investigation** — Root cause analysis within 4 hours
4. **Disposition** — Rework, downgrade, or reject
5. **CAPA** — Corrective and preventive action within 48 hours
6. **Verification** — Effectiveness check within 30 days
