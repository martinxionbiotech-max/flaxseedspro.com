# Packaging & Storage

**CCP-5: Metal Detection & Package Integrity**

---

## Process Objective

Package finished products in formats that preserve quality, provide accurate weights, and ensure tamper-evident, traceable delivery to customers.

## Equipment Configuration

```
Product Silo → VFFS / Bag Filler → Weight Check → N₂ Flush → Metal Detector → X-Ray → Date Code → Palletizer → Stretch Wrapper
```

## Packaging Equipment

| Unit | Type | Capacity | Accuracy | Application |
|:-----|:-----|:--------:|:--------:|:------------|
| VFFS | Vertical form-fill-seal | 600 bph | ± 2 g | 10–25 kg bags |
| Bag filler | Open-mouth | 300 bph | ± 10 g | 25 kg multi-ply |
| Bulk filler | Gravity + scale | 40 MT/hr | ± 0.5 kg | 1 MT totes |
| Drum filler | Bottom-up | 20 drums/hr | ± 50 g | 200 L oil drums |
| N₂ flush tunnel | Inertisation | 600 bph | O₂ < 2% | Oxidative-sensitive products |
| Checkweigher | Dynamic | 600 bph | ± 0.5 g | Reject out-of-tolerance |
| Metal detector | Ferrous / NF / SS | 600 bph | Fe 0.5, NF 0.7, SS 1.0 mm | Reject on detection |

## CCP-5 Monitoring

| Check | Frequency | Critical Limit | Action |
|:------|:---------:|:--------------|:-------|
| Metal detector test | Every 30 min | All 3 test wands detected | Stop line, quarantine |
| N₂ residual O₂ | Every bag (inline) | < 2% | Reject bag, check N₂ supply |
| Checkweigher | Every bag | Per spec ± tolerance | Reject, calibrate |
| Seal integrity | Every 60 min | Visual + burst test | Adjust sealer jaws |
