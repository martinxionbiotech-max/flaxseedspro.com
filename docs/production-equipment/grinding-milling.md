# Grinding & Milling

---

## Process Objective

Reduce flaxseed kernel or whole seed to controlled particle sizes for powder production, with optional cryogenic grinding for heat-sensitive applications.

## Equipment Configuration

```
Feed → Metal Detector → Grinding Mill → Pneumatic Convey → Sifter → Cyclone Collector → Powder Silo
```

## Milling Technologies

| Mill Type | RPM Range | Screen | Throughput | Heat Rise | Application |
|:----------|:---------:|:------:|:----------:|:---------:|:------------|
| Hammer mill | 2800–3600 | 0.8–3.0 mm | 1.0–2.0 MT/hr | 10–15°C | Standard grinding |
| Disc mill | 1200–1800 | Adjustable gap | 0.5–1.0 MT/hr | 8–12°C | Coarse grinding |
| Pin mill | 6000–12000 | Fixed | 0.3–0.8 MT/hr | 12–18°C | Ultra-fine powder |
| Cryogenic mill | 3000–4500 | 0.5–2.0 mm | 0.2–0.5 MT/hr | –150°C | Full-fat, heat-sensitive |

## Cryogenic Grinding

| Parameter | Value |
|:----------|:------|
| Liquid N₂ consumption | 1.5–2.5 kg N₂ / kg product |
| Product temperature | –80 to –150°C |
| Oil retention | 98% of native (vs. 92% at ambient) |
| Peroxide value (fresh) | < 1.0 meq/kg (vs. < 2.0 meq/kg ambient) |
| Particle size | D50 30–80 µm (hammer: 200–400 µm) |
