# Dehulling System

---

## Overview

Our mechanical dehulling system removes the tough seed coat from flaxseeds, producing hull and kernel fractions for specialized applications. The system uses impact dehulling followed by air classification.

## Equipment List

| Equipment | Model | Capacity | Purpose |
|:----------|:-----:|:--------:|:--------|
| Impact Dehuller | ID-750 | 3 MT/hr | Seed coat fracture |
| Aspiration Classifier | AC-500 | 3 MT/hr | Hull/kernel separation |
| Hull Grinder | HG-200 | 500 kg/hr | Hull particle reduction |
| Kernel Polisher | KP-300 | 2 MT/hr | Residual hull removal |

## Process Flow

```
Clean Seed → Impact Dehuller → Aspiration Classifier → Hull Stream & Kernel Stream
```

## Applications

- **Kernel fraction**: Higher oil content, used for premium oil pressing
- **Hull fraction**: Rich in lignans, used for dietary fiber supplements

## Key Specs

- Dehulling efficiency: 85-90%
- Kernel purity: ≥ 95%
- Hull purity: ≥ 92%
- Throughput: 3 MT/hr
