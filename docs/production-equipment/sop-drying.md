# SOP-DRYING: Flaxseed Pre-Drying & Fluidised Bed / Microwave Drying

**Document No:** SOP-DR-001  
**Version:** 1.0  
**Effective Date:** 2026-07-20  
**Department:** Production  
**Scope:** This procedure covers the operation of pre-drying, fluidised bed drying, and microwave drying stages for flaxseed processing. Includes startup sequences, temperature monitoring, fire risk management, shutdown, cleaning, and troubleshooting.

---

## 1. Safety

### 1.1 General Safety
- Only trained and authorised personnel may operate drying equipment.
- Ensure all guarding, interlocks, and safety devices are functional before startup.
- Hot surfaces: signage posted on all drying units. Burn risk.
- Lockout/tagout (LOTO) required before any maintenance or cleaning access.

### 1.2 Fire Risk Management
- **Flaxseed dust is combustible.** Dust accumulation must be controlled.
- Housekeeping: daily vacuuming of all horizontal surfaces in drying area. Use explosion-proof vacuum cleaners.
- **Smoke detectors:** Located at dryer inlet, outlet, and exhaust duct. Test weekly.
- **CO₂ extinguishers:** Located at each dryer station (minimum 2 per unit). Monthly inspection.
- **Automatic fire suppression:** CO₂ flood system for microwave dryer cavity. Verify system pressure weekly.
- **Hot work permit** required within 10 m of any drying equipment.

### 1.3 Personal Protective Equipment (PPE)
- Safety glasses, heat-resistant gloves (for handling hot product), hearing protection.
- Dust mask (N95 or P2) when cleaning or handling dry seed.
- FR (flame-resistant) coveralls in the drying area.

### 1.4 Emergency Shutdown
- Emergency stop buttons at dryer control panels and near exhaust fans.
- In case of fire: press E-stop, activate CO₂ suppression, evacuate, call emergency.
- **Do not use water** on electrical or dust fires — use CO₂ or dry chemical.

## 2. Pre-Dryer Operation

### 2.1 Pre-Dryer Description
The pre-dryer is a rotary or belt-style unit that reduces surface moisture from washed flaxseed prior to the main drying stages.

### 2.2 Pre-Dryer Startup
1. Verify pre-dryer drum/belt is clear of debris.
2. Check exhaust fan dampers are open.
3. Start **exhaust fan** at 50% speed.
4. Start **main drive** at minimum speed (2–5 RPM for rotary, 0.5 m/min for belt).
5. Start **air heater** (gas or electric):
   - Gas: purge for 3 minutes, verify flame, ramp to setpoint.
   - Electric: verify contactors, ramp power gradually.
6. Set inlet air temperature to 120°C (do not exceed 130°C).
7. Allow 5 minutes for thermal stabilisation.
8. **Start feed:** Begin feed at 50% rate; ramp to 100% over 5 minutes.
9. Verify **outlet moisture** (target 12–15% for pre-dried seed).

### 2.3 Pre-Dryer Shutdown
1. Stop feed.
2. Reduce heater setpoint to 50°C.
3. Continue running main drive and exhaust fan for 10 minutes (cool-down).
4. Turn off heater.
5. Run exhaust fan for additional 10 minutes.
6. Stop main drive.
7. Stop exhaust fan.
8. Close dampers.

## 3. Fluidised Bed Dryer (FBD) Operation

### 3.1 Description
FBD uses heated air flowing through a perforated bed to fluidise and dry flaxseed. Critical parameters: bed depth, air velocity, inlet temperature.

### 3.2 Pre-Start Checks
- Verify bed plate is clean and all perforations are open.
- Check air distribution manifold for blockages.
- Confirm that dust collection filters (baghouse/cyclone) are clean.
- Check rotary valve / product discharge is functional.
- Verify interlocks (bed temperature, air flow, product level).

### 3.3 Startup Sequence
1. **Start exhaust fan** — ramp to 80% speed.
2. **Start inlet air fan** — ramp to 50% speed.
3. **Start air heater** — set inlet temperature to 90°C.
4. Establish stable airflow with empty bed for 5 minutes.
5. Verify differential pressure across bed: 15–25 mbar.
6. **Set bed depth** target: 200–300 mm.
7. **Feed product:** Start at minimum feed rate.
8. Adjust air velocity to achieve visible fluidisation (typically 1.5–2.5 m/s).
9. Adjust inlet temperature to maintain **outlet temperature at 50–55°C**.
10. Monitor outlet moisture: target 8–10%.

### 3.4 Air Velocity Control
- **Minimum fluidisation velocity:** 1.0 m/s (below this, no fluidisation).
- **Operating velocity:** 1.5–2.5 m/s depending on particle size and moisture.
- Check bed stability: uniform bubbling, no large channels or spouting.
- Adjust via variable frequency drive (VFD) on inlet fan.

### 3.5 Bed Depth Management
- **Target depth:** 200–300 mm (measured by differential pressure or sight glass).
- If depth <150 mm: increase feed rate or reduce discharge rate.
- If depth >350 mm: increase discharge rate or reduce feed — risk of plugging.

### 3.6 Shutdown
1. Reduce feed rate to zero.
2. Allow bed to empty (run for 2–3 minutes after feed stops).
3. Reduce inlet temperature to 40°C.
4. Reduce air velocity to 50% for 5 minutes (cool-down).
5. Turn off heater.
6. Continue running fans until outlet temperature is <35°C.
7. Stop inlet fan.
8. Stop exhaust fan.

## 4. Microwave Dryer Operation

### 4.1 Description
Finish drying using microwave energy for precise moisture control. Belt speed and power ramp are critical parameters.

### 4.2 Pre-Start Checks
- Verify microwave cavity is clean and free of metal objects.
- Check belt tension and alignment.
- Verify interlock integrity (all cavity doors must close fully).
- Test smoke detector and CO₂ suppression system.
- Confirm product feed system is clear.

### 4.3 Startup Sequence
1. **Turn on belt drive** at minimum speed (0.1 m/min).
2. **Start cavity exhaust fan** (100% speed).
3. **Verify no objects in cavity** via visual inspection through viewing ports.
4. **Power ramp — do not start at full power.**
   - Start at 25% power level.
   - Allow 2 minutes stabilisation.
   - Increase to 50% power — check product temperature.
   - Increase to 75% power — check again.
   - Increase to 100% power only if needed.
5. **Set belt speed:** Adjust to achieve target residence time (typically 2–5 minutes).
6. Start product feed at minimum rate.
7. Verify outlet temperature (target 60–70°C, max 80°C).
8. Measure outlet moisture (target 6–8%).

### 4.4 Power and Belt Speed Correlation
- Higher power + slower belt = more drying, higher temperature.
- Lower power + faster belt = less drying, lower temperature.
- Adjust one parameter at a time. Allow 2 minutes stabilisation between changes.

### 4.5 Shutdown
1. Stop product feed.
2. Set microwave power to 0%.
3. Continue belt drive until cavity is empty (run for belt travel time × 1.5).
4. Turn off belt drive.
5. Run cavity exhaust fan for 5 minutes (cool-down and vapour purge).
6. Turn off exhaust fan.
7. Visually inspect cavity — note any product residue or damage.

## 5. Temperature Monitoring

### 5.1 Instrumentation
- **Pre-dryer:** Inlet air temp, outlet air temp, product discharge temp.
- **FBD:** Inlet plenum temp, bed temp (3 points: entry, middle, exit), exhaust temp.
- **Microwave:** IR sensor at discharge, fibre-optic probe (if available) for internal temp.

### 5.2 Monitoring Frequency
- Log all temperatures every 15 minutes on **DRY-LOG-001**.
- If any reading deviates >5°C from setpoint, investigate immediately.
- Calibrate all temperature sensors monthly using a certified reference.

### 5.3 Over-Temperature Response
| Equipment | Temp Limit | Action |
|---|---|---|
| Pre-dryer | Outlet >70°C | Reduce heater setpoint |
| FBD | Bed >65°C | Reduce inlet temp or increase feed rate |
| Microwave | Discharge >85°C | Reduce power or increase belt speed |
| Any unit | Smoke detected | Emergency shutdown (see Section 1.4) |

## 6. Fire and Smoke Emergency Response

### 6.1 Smoke Detection Alarm
1. Immediate: press **E-stop** on all affected dryers.
2. Activate **CO₂ suppression** system for affected unit.
3. Evacuate the area.
4. Close fire dampers in ductwork (if manual).
5. Call emergency response (ext. 5555).
6. **Do not open** equipment doors until fire department confirms it is safe.
7. Do not re-enter for at least 30 minutes after CO₂ release.

### 6.2 Post-Incident
- Do not restart until fire marshal and maintenance inspection completed.
- Replace all damaged components.
- Clean all ductwork and filters.
- Retest all safety systems before restart.

## 7. Cleaning Procedures

### 7.1 Between Product Changeovers
1. Clear all product from feed hopper and conveying system.
2. **FBD:** Vacuum bed plate surface. Remove bed plate and clean perforations with compressed air. Clean baghouse filters via pulse-jet cycle (5 cycles).
3. **Microwave:** Wipe cavity interior with damp cloth. Remove belt and clean with hot water and mild detergent. Inspect waveguide covers for damage.
4. **Pre-dryer:** Run empty with low heat (60°C) for 10 minutes to dry out residual moisture. Scrape any baked-on material from drum/belt surface.

### 7.2 Deep Cleaning (Weekly)
1. Full LOTO on all equipment.
2. Remove and wash all filters.
3. Clean all ducting with vacuum and brush.
4. Inspect and clean temperature sensor probes.
5. Check and clean CO₂ suppression nozzles.

## 8. Troubleshooting

| Problem | Possible Cause | Corrective Action |
|---|---|---|
| **Uneven moisture (wet spots)** | FBD: bed depth too high | Reduce bed depth to <300 mm |
|  | FBD: channelling in bed | Reduce air velocity; check for blockages in bed plate |
|  | Microwave: power distribution uneven | Verify waveguide/launcher condition; balance magnetrons |
|  | Pre-dryer: uneven feed distribution | Adjust feed spreader |
| **Product overheating** | Inlet air temperature too high | Reduce heater setpoint |
|  | Feed rate too low | Increase feed rate to provide thermal load |
|  | Residence time too long | Increase belt speed or reduce bed depth |
|  | Microwave power too high | Reduce power level; gradual ramp-down |
|  | Air velocity too low (FBD) | Increase to restore fluidisation |
| **Fire / smoke** | Dust ignition | See Section 6. Housekeeping deficiency — implement daily cleaning |
|  | Hot spot from friction | Check bearings, belt tracking, and seals |
|  | Overheated product | Immediately reduce heat source |
|  | Electrical fault | LOTO; call electrician |
| **Low outlet temperature** | Inlet temp set too low | Increase setpoint |
|  | Feed rate too high | Reduce feed rate |
|  | Excessive air velocity (heat carryover) | Reduce fan speed |
| **High pressure drop (FBD)** | Bed plate perforations blocked | Clean with compressed air or replace plate |
|  | Bed depth too high | Increase discharge rate |
|  | Dust collector filters clogged | Pulse-jet clean; replace if needed |
| **Microwave not powering up** | Door interlock fault | Check magnet switches |
|  | Over-temp trip | Allow to cool; check cooling system |
|  | CO₂ system triggered | Reset fire panel |
|  | Magnetron failure | Call maintenance — replace magnetron |
