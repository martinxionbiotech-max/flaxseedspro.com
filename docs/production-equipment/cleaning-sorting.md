# Cleaning & Sorting Equipment

---

## Overview

Raw flaxseed intake processing begins with our multi-stage cleaning and sorting system, ensuring foreign material removal and size grading before further processing.

## Equipment List

| Equipment | Model | Capacity | Purpose |
|:----------|:-----:|:--------:|:--------|
| Gravity Separator | GS-2000 | 5 MT/hr | Density separation |
| Optical Sorter | OS-1500 | 3 MT/hr | Color-based defect removal |
| Aspirator | ASP-1000 | 8 MT/hr | Light material removal |
| Magnetic Separator | MS-500 | 8 MT/hr | Ferrous metal removal |
| Rotary Sieve | RS-800 | 5 MT/hr | Size classification |

## Process Flow

```
Raw Intake → Aspirator → Gravity Separator → Rotary Sieve → Optical Sorter → Magnetic Separator → Clean Seed Silo
```

## Key Specs

- Total throughput: 8 MT/hr
- Purity output: ≥ 99.9%
- Foreign material reduction: > 99%
- Yield loss: < 1%
