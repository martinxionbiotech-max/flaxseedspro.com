# Oil Pressing

**CCP-2: Temperature Control & Solvent Recovery**

---

## Process Objective

Extract flaxseed oil from kernel while preserving native fatty acid profile and minimising oxidative degradation. Two routes: cold pressing and solvent extraction.

## Equipment Configuration

### Cold Pressing Route

```
Kernel → Screw Press (T < 40°C) → Crude Oil → Plate Filter → Polish Filter → N₂ Flush → Packaging
```

### Solvent Extraction Route (defatted meal)

```
Cold-Pressed Cake → Flaker → Hexane Extractor → Miscella → Evaporator → Crude Oil
                                              → Defatted Meal → Desolventiser → Meal
```

## Process Parameters (Cold Press)

| Parameter | Set Point | Tolerance |
|:----------|:---------:|:---------:|
| Kernel temperature at feed | 30°C | ± 2°C |
| Barrel temperature (max) | 40°C | absolute |
| Screw speed | 15 RPM | ± 2 RPM |
| Nozzle diameter | 8 mm | ± 0.5 mm |
| Oil yield (pressing) | 32–38% | — |
| Residual oil in cake | 8–12% | — |

## Solvent Extraction

| Parameter | Value | Unit |
|:----------|:-----:|:-----|
| Hexane temperature | 55–60 | °C |
| Extraction time | 45–60 | min |
| Miscella concentration | 20–25 | % oil |
| Residual oil (defatted meal) | < 1.5 | % |
| Hexane loss | < 0.5 | L/MT meal |

## Quality Monitoring (CCP-2)

| Check | Frequency | Critical Limit | Action |
|:------|:---------:|:--------------|:-------|
| Press temperature | Continuous | < 40°C | Reduce feed rate or stop |
| Peroxide value | Every batch | ≤ 2.0 meq/kg | Hold for review |
| Residual hexane (meal) | Every batch | < 50 ppm | Rerun desolventisation |
