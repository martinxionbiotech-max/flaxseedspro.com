# Automation & Control Systems

---

## Overview

Our facility is managed by a distributed control system providing real-time monitoring, automated batch tracking, and comprehensive data logging for traceability and quality assurance.

## Systems

| System | Vendor/Model | Function |
|:-------|:------------:|:---------|
| SCADA | Wonderware | Process visualization & control |
| PLC | Allen-Bradley ControlLogix | Equipment automation |
| MES | Custom | Batch tracking & lot genealogy |
| LIMS | LabWare | Laboratory data management |
| ERP | SAP Business One | Inventory & order management |

## Automation Features

- **Recipe Management**: Pre-set processing parameters for each product
- **Batch Tracking**: Full lot traceability from raw material to shipment
- **Alarm Management**: Real-time alerts for out-of-spec conditions
- **Reporting**: Automated COA (Certificate of Analysis) generation
- **Integration**: LIMS data automatically linked to batch records

## Monitoring Points

| Stage | Monitored Parameters |
|:------|:---------------------|
| Receiving | Moisture, temperature, weight, origin |
| Cleaning | Power draw, vibration, airflow |
| Milling | Motor current, temperature, RPM |
| Oil Extraction | Temperature, pressure, flow rate |
| Packaging | Weight accuracy, seal integrity, O₂ level |
